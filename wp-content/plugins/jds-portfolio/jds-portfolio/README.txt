JDs Portfolio Plugin enables you to Add a projects detailed and produce portfolio page to display information of past Projects.

== Description ==

While using the Portfolio plugin you possibly can create an original page for displaying stock portfolio items comprising screenshot and extra information.

JDs portfolio uses 5 differnt coloration, Fade in addition to Slide Hover animation, Mixitup jQuery routing effect costomization utilizing general selections.

= Features =

 * Admin Panel for Customization
 * Fully Responsive Portfolio
 * Multy Columns Layout
 * 5 Awesome Color Design
 * Slide and Fade Hover Animation
 * Custom Image Height/Width
 * Category Navigation
 * Custom Portfolio Redirection Link
 * Recent Portfolio widget
 * Admin popup for add shortcode

== Installation ==

1. Upload 'JDs_portfolio' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the new menu item "JDs Portfolio" and create your first Portfolio
4. use "[JDs_portfolio]" in your page to show your Work!
